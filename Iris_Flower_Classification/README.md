The Iris dataset was analyzed using Exploratory Data Analysis and visualizations.

Petal Length and Petal Width were identified as the most important features for distinguishing flower species.

Two machine learning models were trained:

1. Logistic Regression
2. K-Nearest Neighbors (KNN)

Both models achieved high accuracy. KNN provided the best performance on the test dataset and was selected as the final model.

The project demonstrates the complete machine learning workflow including data loading, EDA, visualization, model training, evaluation, and model comparison.
