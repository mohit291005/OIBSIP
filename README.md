# OIBSIP - Oasis Infobyte Data Science Internship

This repository contains the projects completed as part of the Oasis Infobyte Data Science Internship Program.

## Projects

### 1. Iris Flower Classification
**Objective:** Predict the species of an Iris flower using machine learning classification techniques.

**Technologies Used:**
- Python
- Pandas
- Scikit-Learn
- Matplotlib
- Seaborn

**Models Used:**
- Logistic Regression
- K-Nearest Neighbors (KNN)

---

### 2. Car Price Prediction with Machine Learning
**Objective:** Predict the selling price of a used car based on factors such as car age, fuel type, transmission, ownership, and kilometers driven.

**Technologies Used:**
- Python
- Pandas
- Scikit-Learn
- Matplotlib
- Seaborn

**Models Used:**
- Linear Regression
- Random Forest Regressor

---

### 3. Sales Prediction Using Python
**Objective:** Predict product sales based on advertising expenditures across TV, Radio, and Newspaper channels.

**Technologies Used:**
- Python
- Pandas
- Scikit-Learn
- Matplotlib
- Seaborn

**Models Used:**
- Linear Regression
- Random Forest Regressor

---
